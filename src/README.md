# Blogit API

A secure and scalable user authentication API built with **Express.js**, **Prisma ORM**, and **TypeScript**. This backend service handles user registration, login, and logout with robust validation and token-based authentication.

# Features

- User registration with password hashing
- Login via email or username
- JWT-based authentication with cookie support
- Logout by clearing auth token
- Middleware for:
  - Input validation
  - Unique email/username checks
  - Password strength enforcement

# Tech Stack

- **Express.js** – Routing and middleware
- **Prisma ORM** – Database access
- **TypeScript** – Type safety
- **JWT** – Authentication
- **bcryptjs** – Password hashing
- **dotenv** – Environment variable management
